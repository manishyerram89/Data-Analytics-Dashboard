# Data Analytics Dashboard

This dashboard provides a comprehensive overview of sales, profit, cost analysis, and geospatial business performance. It is designed to help organizations monitor financial metrics, analyze product and category performance, evaluate regional activity, and understand cost distribution across promotion types and manufacturers.
